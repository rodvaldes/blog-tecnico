# blog-tecnico
---

## Introducción

Este repositorio posee el contenido estático de un blog técnico basado en jekyll y 
GitHub pages.

## Instrucciones

```bash
$ bundle exec jekyll new . --force
$ bundle exec jekyll serve
```
